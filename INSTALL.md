# 📦 Установка Space Music Hub (Windows)

Полноценное десктоп-приложение: скачал установщик → запустил → графический
**мастер настройки** задаёт пару вопросов → дальше всем управляет неоновый
дашборд. **Python ставить не нужно. Файлы вручную править не нужно.**

---

## Что понадобится

1. **Бот Telegram** — бесплатно создаётся за минуту у [@BotFather](https://t.me/BotFather)
2. **Telegram-канал(ы)**, куда бот будет публиковать музыку
3. **Плейлист(ы) YouTube**, за которыми бот следит

---

## Шаг 1. Скачай и установи

1. Скачай **`SpaceMusicHub-Setup-v2.0.0.exe`** со страницы
   **[Releases](https://github.com/AAvlasins-dev/Music-from-Youtube-playlist-to-telegram/releases/latest)**
2. Запусти установщик и пройди пару экранов (язык → папка → установить).
3. Запусти **Space Music Hub** из меню «Пуск» или с рабочего стола.

> ffmpeg для конвертации в MP3 уже встроен в приложение — отдельно ничего
> качать не нужно.

---

## Шаг 2. Создай бота в Telegram

1. Открой **[@BotFather](https://t.me/BotFather)** → отправь `/newbot`
2. Придумай **имя** и **username** бота (username должен заканчиваться на `bot`)
3. BotFather пришлёт **токен** — длинную строку вида `8280083661:AAE7tJ...`
   Скопируй её целиком, она понадобится в мастере.

> ⚠️ Никому не показывай токен — кто им владеет, тот управляет ботом.

---

## Шаг 3. Сделай бота админом канала

Для **каждого** канала:

1. Канал → **Управление каналом** → **Администраторы**
2. **Добавить администратора** → найди своего бота
3. Включи права:
   - ✅ **Публикация сообщений**
   - ✅ **Закрепление сообщений**

> Без прав админа бот не сможет публиковать треки.

---

## Шаг 4. Мастер настройки

При первом запуске откроется графический мастер из трёх шагов
**TOKEN → CHANNEL → DONE**:

1. **TOKEN** — вставь токен из BotFather. Кнопка проверки покажет
   `OK — connected`, если токен рабочий.
2. **CHANNEL** — укажи `@имя_канала` и **ссылку на плейлист YouTube**
   целиком. Кнопка **«Тест канала»** проверит, что бот видит канал и
   считает треки (`OK — 1055 tracks`). Можно добавить несколько пар
   канал + плейлист.
3. **DONE** — настройки сохраняются, и можно сразу запустить.

Ссылку на плейлист вставляй целиком — приложение само достанет ID:
```
https://www.youtube.com/playlist?list=PLeaNX9v3Xr40bmBwZD7lUQfaRNSufJ_hF
```

## Шаг 5. Дашборд — как всё работает

После настройки открывается дашборд с кнопками:

| Кнопка | Что делает |
|---|---|
| ▶ **WATCH** | главный режим — проверяет плейлист по расписанию и сам публикует новые треки |
| ⚡ **RUN ONCE** | один проход: опубликовать всё новое и остановиться |
| 🔍 **CHECK** | только проверить настройки и посчитать новое, ничего не качая |
| ⏹ **STOP** | остановить текущий запуск |
| ⚙ **CONFIG** | изменить токен, каналы или плейлисты (мастер заново) |

**Расписание (CHECK SCHEDULE).** Выбери, как часто проверять плейлист:
**каждый час / каждые 2 часа / раз в день / раз в неделю**. В режиме
**WATCH** приложение проверяет с этим интервалом и продолжает работать.
Кнопка **Check now** делает один проход прямо сейчас.

**Статистика** показывает, сколько треков опубликовано (POSTED), сколько
с ошибкой (FAILED) и сколько запусков было (RUNS). Внизу — живой **лог**
(переключатель Simple/Expert).

> 💡 **Первый запуск долгий** — если в плейлисте 1000+ треков, публикация
> займёт несколько часов. Прогресс сохраняется после каждого трека:
> приложение можно закрыть и продолжить позже — оно не начнёт заново.

## Шаг 6. Автозапуск в фоне (трей)

Хочешь, чтобы бот работал сам, без открытого окна?

Поставь галочку **«Launch at Windows startup (in tray)»** на дашборде.
Тогда приложение будет:

- запускаться при входе в Windows,
- сворачиваться в **системный трей** (значок у часов),
- проверять плейлист по выбранному расписанию и публиковать новое.

Открыть окно обратно — клик по значку в трее. Снять галочку — выключить
автозапуск.

## ❓ Что-то пошло не так

| Проблема | Решение |
|---|---|
| «Токен неверный» | Токен введён с ошибкой — скопируй заново у @BotFather |
| «канал пока не виден» | Бот не добавлен в канал как админ, или канал приватный — исправь и нажми **Тест канала** ещё раз |
| Треки не приходят | Проверь, что бот — **администратор** канала с правом публикации |
| `[Errno 22]` при скачивании | Путь скачивания содержит кириллицу. В **CONFIG** оставь `DOWNLOAD_DIR=C:\Temp\music_bot` |
| «программа уже запущена» | Приложение уже работает. Если зависло — удали файл `bot.lock` в папке установки |

Все действия пишутся в `bot.log` в папке установки (токен в логи **не**
попадает — он маскируется).

## 🛠 Собрать `.exe` из исходного кода (для разработчиков)

Нужен Python 3.12 и [Inno Setup 6](https://jrsoftware.org/isdl.php) (для установщика).

```cmd
pip install -r requirements-dev.txt
build_exe.bat
```

`build_exe.bat` запускает PyInstaller со спеком `SpaceMusicHubGUI.spec` и
собирает приложение в `dist\SpaceMusicHub\SpaceMusicHub.exe`. Чтобы
упаковать установщик, скомпилируй `installer\SpaceMusicHub.iss` в Inno Setup.

Подробности процесса сборки — в [README](README.md#-build-from-source).
