# 📦 Установка Space Music Hub (Windows)

Полноценное приложение: скачал → запустил → мастер настройки задаёт пару вопросов → готово.
**Python устанавливать не нужно.** Редактировать файлы вручную тоже не нужно.

---

## Что понадобится

1. **Бот Telegram** — бесплатно создаётся за минуту у [@BotFather](https://t.me/BotFather)
2. **Telegram-канал(ы)**, куда бот будет публиковать музыку
3. **Плейлист(ы) YouTube**, за которыми бот следит

---

## Шаг 1. Скачай и распакуй

1. Скачай `SpaceMusicHub-win64.zip` со страницы **[Releases](https://github.com/AAvlasins-dev/Music-from-Youtube-playlist-to-telegram/releases)**
2. Распакуй в любую папку, например `C:\SpaceMusicHub\`

Внутри:
```
C:\SpaceMusicHub\
├── SpaceMusicHub.exe   ← сама программа
├── ffmpeg.exe          ← конвертация в MP3
└── INSTALL.md          ← эта инструкция
```

---

## Шаг 2. Создай бота в Telegram

1. Открой **[@BotFather](https://t.me/BotFather)** → отправь `/newbot`
2. Придумай имя и username бота
3. BotFather пришлёт **токен** — длинную строку вида `8280083661:AAE7tJ...`
   Скопируй её, она понадобится в мастере.

---

## Шаг 3. Сделай бота админом канала

Для **каждого** канала:

1. Канал → **Управление каналом** → **Администраторы**
2. **Добавить администратора** → найди своего бота
3. Включи права:
   - ✅ **Публикация сообщений**
   - ✅ **Закрепление сообщений**

> Без прав админа бот не сможет публиковать треки.

---

## Шаг 4. Запусти программу — мастер настройки

Просто **дважды кликни на `SpaceMusicHub.exe`**.

При первом запуске откроется **мастер настройки**, который сам всё спросит:

```
[1/3] Telegram bot token
  Paste bot token:  ← вставь токен из BotFather (Ctrl+V)
  OK - connected as @твой_бот     ← программа проверит токен

[2/3] Channels
  Telegram channel (@name or t.me/name):  ← напиши @имя_канала
  OK - found channel: Название              ← программа проверит канал
  YouTube playlist URL or ID:  ← вставь ссылку на плейлист целиком
  OK - 1055 tracks found                    ← программа посчитает треки
  Add another channel? (y/N):  ← y если нужен ещё канал, иначе Enter

[3/3] Saving configuration...
  Saved: C:\SpaceMusicHub\.env
Setup complete!

Run the bot now? (Y/n):  ← нажми Enter чтобы запустить сразу
```

**Ссылку на плейлист** можно вставлять целиком — программа сама достанет ID:
```
https://www.youtube.com/playlist?list=PLeaNX9v3Xr40bmBwZD7lUQfaRNSufJ_hF
```

Вот и всё! Никаких файлов вручную править не нужно.

---

## Шаг 5. Главное меню

После настройки при каждом запуске программа показывает меню:

```
  SPACE MUSIC HUB
  1) Запустить (авто)  — следить и публиковать новые треки
  2) Запустить 1 раз   — опубликовать новые и выйти
  3) Проверка          — проверить настройки (без публикации)
  4) Перенастроить     — пройти мастер заново
  5) Выход
  Выбор (1-5):
```

- **1 — Запустить (авто)** ⭐ — главный режим. Программа работает постоянно:
  пока **окно открыто**, она каждые 15 минут проверяет плейлист и сама
  публикует новые треки. Просто **оставь окно открытым** — и музыка прилетает.
- **2 — Запустить 1 раз** — опубликовать новые треки и закрыться.
- **3 — Проверка** — всё ли настроено, сколько новых треков (ничего не публикует).
- **4 — Перенастроить** — мастер заново (например, добавить ещё канал).

> 💡 **Первый запуск долгий** — если в плейлисте 1000+ треков, публикация займёт
> несколько часов. Прогресс сохраняется после каждого трека: окно можно закрыть
> и продолжить позже — программа не начнёт заново.

## Шаг 6. Просто оставь окно открытым

Для брата/обычного пользователя ничего больше настраивать не нужно:

1. Дважды кликни `SpaceMusicHub.exe`
2. Выбери **1) Запустить (авто)** (или нажми Enter сразу после мастера)
3. **Оставь чёрное окно открытым** — пока компьютер включён и окно открыто,
   новые треки сами появляются в канале.

Чтобы остановить — просто закрой окно.

### (Необязательно) Автозапуск без окна — Планировщик Windows

Если хочешь, чтобы бот проверял треки сам по расписанию (без открытого окна):

1. **Win** → «**Планировщик заданий**» → **Создать задачу**
2. **Триггеры** → **Создать** → **Ежедневно**, например `10:00`
3. **Действия** → **Запуск программы**: путь к `SpaceMusicHub.exe`,
   **аргументы:** `--run`   ← запускает один проход без меню
4. **Параметры**: отметь «Выполнить задачу как можно скорее после пропуска» → **ОК**

> ⚠️ Для Планировщика обязательно аргумент **`--run`** — иначе программа покажет
> меню и будет ждать. (Аргумент `--watch` — постоянная работа без меню.)

## ❓ Что-то пошло не так

| Проблема | Решение |
|---|---|
| Окно мелькнуло и закрылось | Запусти через командную строку чтобы увидеть текст, или смотри `bot.log` рядом с программой |
| «Токен неверный» | Токен введён с ошибкой — скопируй заново у @BotFather |
| «канал пока не виден» | Бот ещё не добавлен в канал как админ, или канал не публичный — исправь и пройди мастер заново (пункт 4 меню) |
| Треки не приходят | Проверь что бот — **администратор** канала с правом публикации |
| `[Errno 22]` при скачивании | Путь скачивания содержит кириллицу. Перенастрой и оставь `DOWNLOAD_DIR=C:\Temp\music_bot` |
| «программа уже запущена» | Программа уже работает. Если зависла — удали файл `bot.lock` рядом с программой |

Все действия пишутся в `bot.log` рядом с программой.

## 🔧 Командная строка (для опытных)

```cmd
SpaceMusicHub.exe           :: меню (или мастер при первом запуске)
SpaceMusicHub.exe --watch   :: постоянная работа без меню (треки прилетают сами)
SpaceMusicHub.exe --run     :: один проход без меню (для Планировщика)
SpaceMusicHub.exe --check   :: проверка без публикации
SpaceMusicHub.exe --setup   :: перенастроить заново
```
