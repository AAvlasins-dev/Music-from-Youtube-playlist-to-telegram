# 📦 Установка Space Music Hub (Windows, .exe)

Подробная инструкция: как установить и настроить бота на компьютере **без установки Python**.

---

## Что понадобится

1. **Бот Telegram** — токен от [@BotFather](https://t.me/BotFather)
2. **Telegram-каналы**, куда бот будет слать музыку (ты должен быть их админом)
3. **Плейлисты YouTube**, за которыми бот будет следить
4. Файлы из релиза:
   - `SpaceMusicHub.exe` — сам бот
   - `ffmpeg.exe` — конвертация в MP3
   - `.env` — твои настройки

---

## Шаг 1. Создай бота и получи токен

1. Открой Telegram → напиши **[@BotFather](https://t.me/BotFather)**
2. Отправь команду `/newbot` (или `/mybots` → выбери существующего)
3. Скопируй **API Token** — выглядит так:
   ```
   8280083661:AAE7tJTovxEOl-Iv43aooeggT2-CeH8pQZc
   ```

---

## Шаг 2. Сделай бота админом в каналах

Для **каждого** канала:

1. Открой канал → **Управление каналом** → **Администраторы**
2. **Добавить администратора** → найди своего бота по имени
3. Выдай права:
   - ✅ **Публикация сообщений**
   - ✅ **Закрепление сообщений**

> Без прав админа бот не сможет ничего отправить.

---

## Шаг 3. Узнай ID плейлистов YouTube

ID плейлиста — это часть ссылки после `list=`:

```
https://www.youtube.com/playlist?list=PLeaNX9v3Xr40bmBwZD7lUQfaRNSufJ_hF
                                       └──────────── это ID ────────────┘
```

---

## Шаг 4. Положи файлы в одну папку

Создай папку, например `C:\SpaceMusicHub\`, и положи туда три файла:

```
C:\SpaceMusicHub\
├── SpaceMusicHub.exe
├── ffmpeg.exe
└── .env
```

---

## Шаг 5. Настрой файл `.env`

Открой `.env` в **Блокноте** и заполни:

```ini
# Токен бота от @BotFather
TELEGRAM_BOT_TOKEN=сюда_вставь_свой_токен

# Канал 1: ID плейлиста + имя канала (БЕЗ символа @)
CHANNEL_1_NAME=music
CHANNEL_1_PLAYLIST=PLeaNX9v3Xr40bmBwZD7lUQfaRNSufJ_hF
CHANNEL_1_TELEGRAM=music_ebat_2026

# Канал 2 (если нужен — иначе удали эти три строки)
CHANNEL_2_NAME=baiba
CHANNEL_2_PLAYLIST=FLta_1jnlsUWABxzgqeLFRVQ
CHANNEL_2_TELEGRAM=baiba_music

# Куда временно скачивать MP3 (путь только латиницей, без кириллицы!)
DOWNLOAD_DIR=C:\Temp\music_bot
```

> **Хочешь больше каналов?** Просто добавь `CHANNEL_3_*`, `CHANNEL_4_*` и так далее — менять программу не нужно.

> ⚠️ **Важно:** `DOWNLOAD_DIR` должен быть путём только из латинских букв (например `C:\Temp\music_bot`). Кириллица в пути ломает конвертацию ffmpeg.

Сохрани файл (`Ctrl+S`).

---

## Шаг 6. Проверка перед запуском (`--check`)

Прежде чем запускать «по-настоящему», проверь что всё настроено. Открой **командную строку** в папке с ботом и выполни:

```cmd
SpaceMusicHub.exe --check
```

Бот **ничего не отправит**, только проверит:
- ✅ токен рабочий — покажет имя бота
- ✅ сколько треков в каждом плейлисте
- ✅ сколько новых треков к отправке

Если видишь `Check complete — configuration looks valid.` — всё готово.

> Как открыть командную строку в папке: открой папку в Проводнике → в адресной строке напиши `cmd` → Enter.

---

## Шаг 7. Запуск

Просто **дважды кликни на `SpaceMusicHub.exe`**.

Откроется чёрное окно — бот начнёт качать треки и отправлять их в каналы. Прогресс виден в окне:

```
[music] [1/1055] Posted: Название трека
[music] [2/1055] Posted: Следующий трек
...
```

> 💡 **Первый запуск долгий** — если в плейлисте 1000+ треков, это несколько часов. Бот сохраняет прогресс после каждого трека, так что окно можно закрыть и запустить позже — он продолжит с места остановки.

---

## Шаг 8. Автозапуск раз в день (Планировщик Windows)

Чтобы бот сам проверял новые треки каждый день:

1. Нажми **Win** → набери «**Планировщик заданий**» → открой
2. Справа → **Создать задачу** (не «простую задачу»)
3. Вкладка **Общие**: имя `Space Music Hub`, отметь **Выполнять с наивысшими правами**
4. Вкладка **Триггеры** → **Создать** → **Ежедневно**, время например `10:00`
5. Вкладка **Действия** → **Создать** → **Запуск программы** → укажи путь к `SpaceMusicHub.exe`
6. **ОК**

Теперь бот будет запускаться каждый день в 10:00 и доливать только новые треки.

> Компьютер должен быть включён в это время. Если был выключен — Windows запустит задачу при следующем включении (поставь галочку «Выполнить задачу как можно скорее после пропуска» во вкладке **Параметры**).

---

## ❓ Что-то пошло не так

| Проблема | Решение |
|---|---|
| Окно мелькнуло и закрылось | Запусти через `cmd` чтобы увидеть ошибку, или смотри файл `bot.log` рядом с exe |
| `No channels configured` | Проверь что в `.env` заполнены `CHANNEL_1_PLAYLIST` и `CHANNEL_1_TELEGRAM` |
| `401 Unauthorized` | Неверный токен — возьми новый у @BotFather |
| `Chat not found` | Бот не добавлен в канал, или имя канала указано неверно (без `@`) |
| Треки не приходят | Убедись что бот — **администратор** канала с правом публикации |
| `[Errno 22]` при скачивании | Путь в `DOWNLOAD_DIR` содержит кириллицу — поменяй на `C:\Temp\music_bot` |
| `another instance may be running` | Бот уже запущен. Если завис — удали файл `bot.lock` рядом с exe |

Лог всех действий пишется в `bot.log` рядом с программой — открой его, если нужно разобраться.

## 🔧 Собрать .exe самому (для разработчиков)

Если хочешь собрать `.exe` из исходного кода:

```cmd
pip install -r requirements.txt
pip install pyinstaller
build_exe.bat
```

Готовый файл появится в `dist\SpaceMusicHub.exe`.
