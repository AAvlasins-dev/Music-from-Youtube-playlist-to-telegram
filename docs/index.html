How it works

A simple, reliable pipeline. Runs on a schedule, never posts the same track twice, and resumes exactly where it left off after any interruption.

📋
Fetch the playlist
Reads every video via yt-dlp — no YouTube API key required.
🔍
Filter what's new
Compares against per-channel state so already-posted tracks are skipped.
🎵
Download & convert
yt-dlp + ffmpeg produce a real 192 kbps MP3 file.
📤
Post & pin
Sends the audio to Telegram with the YouTube link, pins the latest track.
💾
Save state after every track
A crash or restart resumes from the last success — never from scratch.
