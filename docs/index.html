Pipeline

How it works

A simple, reliable pipeline. Runs on a schedule, never posts the same track twice, and resumes exactly where it left off after any interruption.

📋
Fetch Playlist
yt-dlp reads every video. No API key needed.
🔍
Filter New
Already-posted tracks are skipped.
🎵
Convert MP3
Real 192 kbps audio via ffmpeg.
📤
Post & Pin
Sends to Telegram + pins the latest.
💾
Save State
Crash or restart? Resumes from last success.
